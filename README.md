# Calculator

## Platform: Android
## IDE: AndroidStudio
This app was created using Android Studio and involves multiple fragments.
It is a basic calculator.

### Functionality
- Numbers 0 - 9
- Multiplication, Division, Subtraction, Addition
- Decimals
- Negative Numbers
- Memory Store
- Memory Recall
- Delete
- Clear
